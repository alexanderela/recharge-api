To get charging station data for a specific station, use the URL:

https://recharge-api.herokuapp.com/api/v1/stations/:station_id

in your fetch request, which returns a response body containing an array which contains the specified station object.

Example URL:


            https://recharge-api.herokuapp.com/api/v1/stations/1

POST Endpoints

To add a charging station to the database, use the URL:

https://recharge-api.herokuapp.com/api/v1/stations/

in your fetch request, which returns a response body with an id and a message.

Example Request:


            {
              "station_name": "Chris' Awesome Station",
              "station_phone": "555-555-5555",
              "street_address": "123 17th",
              "city": "Denver",
              "state": "CO",
              "zip_code": 80220,
              "latitude": 39.75,
              "longitude": -105,
              "intersection_directions": "unknown",
              "access_days_time": "24 hours daily"
            }

Example Response:


            {
              "id": 35,
              "message": "Station \"Chris' awesome station\" successfully created!"
            }

PUT Endpoints

To change the name of a specific charging station, use the URL:

https://recharge-api.herokuapp.com/api/v1/stations/:station_id

in your fetch request, which returns a response body with a message.

Example Request:


            {
              "station_name": "Freddie's awesome station"
            }

Example Response:


            {
              "message": "Edit successful. Station with id of 35 name changed from Chris' awesome station to Freddie's awesome station."
            }

DELETE Endpoints

To delete a specific charging station, use the URL:

https://recharge-api.herokuapp.com/api/v1/stations/:station_id

in your fetch request, which returns a response body with a message and the id of '1' if the delete is successful.

Example URL:

https://recharge-api.herokuapp.com/api/v1/stations/33

Example Response:


            {
              "id": 1,
              "message": "Station 33 has been deleted."
            }

https://recharge-api.herokuapp.com/api/v1/cafes?cafe_name=CAFE+NAME+HERE
